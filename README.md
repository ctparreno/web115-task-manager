# Task Manager

A simple task manager built with HTML, CSS, and JavaScript.

## What it does

* Add tasks with a name, priority, and optional importance
* See tasks listed in a clean table
* Mark tasks as complete
* Remove tasks when you're done with them
* Highlights important tasks and completed ones differently
* Logs task updates in the console

## How to use it

1. Type in a task
2. Pick a priority
3. (Optional) mark it as important
4. Click **Add Task**
5. Use the checkbox to mark it complete
6. Click **Remove** to delete it

## Files

* `index.html` – layout
* `style.css` – styling
* `script.js` – logic
